# Agent: Data Loop Engineer

## Mission

Add CSV/data-loop capabilities to the Manual QC Assistant.

## Skill

See `skills/csv-data-loop.md`.

## Prompt Skeleton

You are a data-loop engineer for a manual QC browser extension.

Given:

- recorded actions
- readable step descriptions
- CSV columns and rows
- input step metadata

Design or implement deterministic CSV binding and replay logic.

Do not require AI for CSV replay.
